# Dashboard de Cobrança

Sistema web desenvolvido para acompanhamento de metas e desempenho da equipe de cobrança.

## Funcionalidades

- Dashboard com indicadores
- Controle de metas semanais e mensais
- Cadastro de usuários
- Ranking de cobradores
- Gráficos de desempenho
- Tema Dark Mode

## Tecnologias

- React
- Vite
- TailwindCSS
- Recharts
- Lucide React

## Futuras Implementações

- API REST
- Banco de dados
- Autenticação
- Relatórios PDF
